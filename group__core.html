Detailed Description

Provides essential utilities and base functionalities for the E2D game engine.

The Core module includes foundational components and services that are fundamental for the E2D game engine. This module offers low-level utilities, basic data structures, and essential interfaces used across the engine. It is designed to be lightweight and efficient, ensuring minimal overhead while providing the necessary building blocks for more complex features in other modules.

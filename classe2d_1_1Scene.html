Detailed Description

Manages a collection of game objects and handles the lifecycle of a game scene.

The Scene encapsulates all the logic and entities that make up a particular section or state of the game. It manages the creation, update, and destruction of game objects, handles input events, and renders the scene to the screen. Each Scene operates independently, allowing the game engine to switch between different scenes seamlessly, such as switching between a menu and gameplay.

Detailed Description

Internal Manages a priority queue of Renderable objects.

RenderQueue is a utility class that manages a queue of Renderable objects, sorting them based on their render priority. This ensures that objects are rendered in the correct order. The class uses a priority queue where objects with higher render priorities are rendered later.

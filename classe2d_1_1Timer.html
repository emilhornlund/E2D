Detailed Description

A class that provides functionality for a timer that can be paused, stopped, or resumed.

The Timer class is designed to measure elapsed time in milliseconds or seconds. It provides methods to start, stop, pause, and resume the timer. Additionally, it offers functionalities to check the timer's current state and retrieve the elapsed time.

Detailed Description

template<typename T>
class e2d::Vector2< T >

Represents a 2-dimensional vector.

The Vector2 class provides functionalities to define and manipulate 2D vectors. It supports basic vector operations such as addition, subtraction, and scaling.

Template Parameters
TThe numerical type of the vector's components (e.g., int, unsigned int, float, double).

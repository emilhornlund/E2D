Detailed Description

Handles the rendering process for graphical objects.

The Renderer class is responsible for managing and executing the rendering process of Renderable objects within the E2D Engine. It maintains a render queue to sort and render objects based on their render priority. The Renderer class provides functions for adding Renderable objects to the queue and for performing the actual rendering process to display the content on the screen.

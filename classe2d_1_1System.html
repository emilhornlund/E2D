Detailed Description

Abstract base class for all engine subsystems.

The System class serves as the abstract base class for all engine subsystems. It defines a standard interface for initialization and shutdown operations that must be implemented by all derived subsystems. This ensures that each subsystem follows a consistent lifecycle, making it easier to manage and integrate into the engine as a whole.

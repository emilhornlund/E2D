Detailed Description

Manages the initialization and shutdown of the engine's core subsystems.

The System class is responsible for setting up and tearing down essential components required for the engine's operation. It provides a unified interface to ensure that these components are correctly prepared before the engine starts and properly released when the engine shuts down.

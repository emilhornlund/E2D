Detailed Description

Manages the lifecycle, transitions, and state of scenes in the engine.

The SceneManager class is responsible for managing scenes within the game engine. It maintains a stack of active scenes, supports scene transitions, and ensures that scenes are properly loaded and unloaded. The SceneManager allows the game to move between different states or levels, such as menus, gameplay, and settings.

const std::shared_ptr<e2d::Scene>& e2d::SceneManager::getActiveScene ( ) const

Retrieves the currently active scene.

This method returns a reference to the scene that is currently active. The active scene is the one at the top of the scene stack. If no scenes are active, an exception is thrown.

◆ popScene()

void e2d::SceneManager::popScene ( )

Pops the current active scene off the stack.

This method unloads and removes the scene at the top of the scene stack. The scene below it (if any) becomes the new active scene. If no scenes are loaded, an exception is thrown.

Exceptions
std::runtime_errorIf no scenes are loaded.

◆ pollEvent()

bool e2d::pollEvent ( e2d::Event &  event)

Polls for an event from the system queue and translates it to an E2D event.

This function checks the system's event queue for new events and, if available, translates them into the E2D engine's internal event format. It is used within the main game loop to process input and other system events.

Parameters
eventReference to an Event struct where the polled event will be stored.
Returns
True if an event was successfully polled, false if the event queue was empty.

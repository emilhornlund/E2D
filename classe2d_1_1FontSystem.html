Detailed Description

Manages the initialization and shutdown of the SDL TTF subsystem.

The FontSystem class is responsible for initializing and shutting down the SDL TTF (TrueType Font) subsystem, which is used for rendering text in different fonts within the engine. This system is essential for any text rendering operations.
